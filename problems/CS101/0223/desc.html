The function H is defined for all real numbers as follows:


H(x) = 0 if x <= 0

H(x) = 1 if 0 < x <= 1

H(x) = 1.25*H(x/2) if x > 1


Write a function called H that takes a single parameter (a float, for x) and returns H(x). The function should do no input or output (no input or print statements, no file access), and may assume that only a float will be passed to it.


Name your file: Function.py
