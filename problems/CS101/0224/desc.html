One problem we sometimes have is taking a large number and adding commas for output; the number
12345678
for example, should print as:
12,345,678.

Suppose we have a string representing an integer > 0. One way of adding commas to it for printing is to use a recursive algorithm:


if StrN <= 3 digits (characters) long:
return StrN
Otherwise
Get a 'comma-fied' version of (StrN with last 3 characters removed)
Add a comma
Add the last 3 digits of StrN
return the result

Write a recursive function using the above algorithm that takes a string representing an integer > 0 and returns a string representing the 'comma-fied' version of the string. The function should do no input or output; its only interaction is through parameters coming in and values being returned. You do not need to verify that the parameter is a string or that it represents an integer.


Name your function AddCommas. Name your file Function.py.
