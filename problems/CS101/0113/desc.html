Word Counts

The object of this program is to count the number of words starting with each letter. The program will use the '>>' prompt to enter the filename. Open the given filename and count the number of words that start with each letter either upper or lower.


The name of the program should be 'solution.py'

Sample run with the sentence 'This is fun!' in the file:

>>Test1.txt
