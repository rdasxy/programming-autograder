One problem we sometimes have is taking a large number and adding commas for output; the number
12345678
for example, should print as:
12,345,678.

Suppose we have a string representing an integer > 0. One way of adding commas to it for printing is to use a recursive algorithm:


if StrN <= 3 digits (characters) long:
return StrN
Otherwise
Get a 'comma-fied' version of (StrN with last 3 characters removed)
Add a comma
Add the last 3 digits of StrN
return the result


Write a recursive function using the above algorithm that takes a string representing an integer > 0 and returns a string representing the 'comma-fied' version of the string. BUT, we're also concerned about malicious users giving us a very long string and trying to force us to do more recursion than our system supports. Therefore, if your function requires more than 5 levels of recursion, it should return the string “Error”. This can then be incorporated into the output as usual; sample output is shown below.


The function should do no input or output; its only interaction is through parameters coming in and values being returned. You do not need to verify that the parameter is a string or that it represents an integer.


Name your function AddCommas. Name your file Function.py.


(Hint: Add a parameter to your function that counts how many times you've called this function. Give it a default value of 5, and count down.)
