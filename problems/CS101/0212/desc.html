Write a function that takes 2 parameters: a dict, and a string (which may or may not be a key in the dicitonary). The function should print the value that has that string for its key. If no such key exists in the dictionary (a KeyError), print: Key not found.


Name your file Function.py

Name the function CheckKey


Although this can be done without a try/except structure, you should use such a structure anyway (the point is to practice this.)


Use this code to test your function:


import Function

D = {'a':'apple', 'b':'banana', 'c':'cherry'}

Function.CheckKey(D, 'a') # should print 'apple'

Function.CheckKey(D, 'f') # should print 'Key not found.'
